# SportFit.com
🌐 Website Prototype

This is a frontend website prototype created to demonstrate layout, design, and basic user interaction.

⚠️ Important: This project is only a prototype. Some features may not work or are only partially implemented.

📌 Description

The project includes a basic website structure built with:

HTML

CSS

JavaScript

It demonstrates UI layout, styling, animations, modals, and basic form interactions.

🛠 Project Structure
/ (root)
│── index.html    # Main page
│── style.css     # Styles
│── index.js      # JavaScript logic
🚀 How to Run

Clone the repository:

git clone <your-repository-url>

Open index.html in your browser.

No additional dependencies or installations are required.

⚙️ Features

✔️ Responsive layout
✔️ Animated UI elements
✔️ Modal windows
✔️ Custom calendar component
✔️ Basic form validation (client-side only)

❌ No backend integration
❌ No real database connection
❌ Forms do not send real requests
❌ Some features are for demonstration purposes only

🎯 Purpose of the Project

Frontend development practice

UI/UX design demonstration

Portfolio project prototype

Base structure for future development
